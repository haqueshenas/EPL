What Is VPL?

The Virtual Phenotyping Lab (VPL) is a non-profit scientific initiative—to develop and share simple, low-cost software tools, code snippets, and computational models for plant and crop phenotyping. Originating from high-throughput phenotyping research by Abbas Haghshenas, VPL invites you to explore, use, and contribute to accessible, reproducible workflows.

Initiated by: Abbas Haghshenas (Independent Researcher in Crop Production, Shiraz, Iran) Contact: haqueshenas@gmail.com

Core Principles

  • Reliability: Rigorous, tested code and scientifically sound methods.
  • Simplicity: Clear examples, minimal dependencies, step-by-step guides.
  • Availability: All code hosted openly with permissive licenses.
  • Affordability: No proprietary software required.
  • Reproducibility: Sample data and parameter files included.
  • Community: Open to any researcher to test and improve.
